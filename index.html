Console Screen

The console screen is specially designed for developers to view necessary values and other details that are not required to be displayed on the main page. For example, while performing long calculations, the programmer can divide the calculation into several parts and print the values obtained in different parts on the console screen, before displaying the final result, to verify the calculation.

We can use the following code to print values on the console screen:

console.log(Enter whatever needs to be printed);

For example, let us display the text "Web App" on the console screen. The required line of code would be- console.log("Web App");

In order to open the console screen, we need to right-click on any blank space of the webpage and click 'Inspect'. We can select the 'Console' option from the menu and view the values that have been printed on the console screen.
